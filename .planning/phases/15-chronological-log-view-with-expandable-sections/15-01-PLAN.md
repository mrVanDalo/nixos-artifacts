<task type="auto">
  <name>Task 3: Implement update handlers for chronological log</name>
  <files>pkgs/artifacts/src/app/update.rs</files>
  <action>
Add match arms in update() for chronological log screen:

1. For `(Screen::ChronologicalLog(mut state), Msg::ToggleSection { step })`:
   - Toggle the section in expanded_sections HashSet
   - Return (Screen::ChronologicalLog(state), Effect::None)

2. For `(Screen::ChronologicalLog(mut state), Msg::ScrollLogs { delta })`:
   - Update scroll_offset, clamp to valid range
   - Return updated state

3. For `(Screen::ChronologicalLog(mut state), Msg::ExpandAllSections)`:
   - Call state.expand_all()
   - Return updated state

4. For `(Screen::ChronologicalLog(mut state), Msg::CollapseAllSections)`:
   - Call state.collapse_all()
   - Return updated state

5. For `(Screen::ChronologicalLog(mut state), Msg::FocusNextSection)`:
   - Cycle focused_section to next LogStep variant
   - Return updated state

6. For `(Screen::ChronologicalLog(mut state), Msg::FocusPreviousSection)`:
   - Cycle focused_section to previous LogStep variant
   - Return updated state

7. Handle Esc key in ChronologicalLog screen to return to ArtifactList.

Pattern: Follow existing screen handler patterns in update.rs.
</action>
