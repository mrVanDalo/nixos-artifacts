<task type="auto">
  <name>Task 2: Implement State Transition Handlers</name>
  <files>pkgs/artifacts/src/app/update.rs</files>
  <action>
    Implement message handlers for all effect result messages:

    1. Handle Msg::CheckSerializationResult:
       ```rust
       Msg::CheckSerializationResult { artifact_index, needs_generation } => {
           let status = if needs_generation {
               ArtifactStatus::NeedsGeneration
           } else {
               ArtifactStatus::UpToDate
           };
           model.set_status(artifact_index, status);
           
           // If needs generation, trigger next effect
           if needs_generation {
               // Check if artifact has prompts
               let artifact = &model.artifacts[artifact_index];
               if artifact.prompts.is_empty() {
                   // No prompts, go straight to generator
                   Effect::RunGenerator { 
                       artifact: artifact.clone(), 
                       prompts: HashMap::new() 
                   }
               } else {
                   // Show prompts - handled by TUI directly per decisions
                   Effect::ShowPrompts { artifact_index }
               }
           } else {
               Effect::None
           }
       }
       ```

    2. Handle Msg::GeneratorFinished:
       ```rust
       Msg::GeneratorFinished { artifact_index, success, output } => {
           if success {
               model.set_status(artifact_index, ArtifactStatus::Serializing);
               // Trigger Serialize effect
               let artifact = &model.artifacts[artifact_index];
               Effect::Serialize { artifact: artifact.clone() }
           } else {
               model.set_status(artifact_index, ArtifactStatus::Failed {
                   error: "Generator failed".to_string(),
                   output,
                   retry_available: true,
               });
               Effect::None
           }
       }
       ```

    3. Handle Msg::SerializeFinished:
       ```rust
       Msg::SerializeFinished { artifact_index, success, error } => {
           let status = if success {
               ArtifactStatus::UpToDate
           } else {
               ArtifactStatus::Failed {
                   error: error.unwrap_or_else(|| "Serialization failed".to_string()),
                   output: String::new(),
                   retry_available: true,
               }
           };
           model.set_status(artifact_index, status);
           Effect::None
       }
       ```

    4. Add handler for generation initiation (Enter key):
       ```rust
       Msg::Key(KeyEvent::Enter) => {
           let selected = model.selected_index;
           if model.can_generate(selected) {
               // Set status to Generating with CheckSerialization step
               model.set_status(selected, ArtifactStatus::Generating(GeneratingState {
                   step: GenerationStep::CheckSerialization,
                   output: String::new(),
               }));
               // Trigger CheckSerialization effect
               let artifact = &model.artifacts[selected];
               Effect::CheckSerialization { artifact: artifact.clone() }
           } else if model.is_generating(selected) {
               // Silently ignore duplicate requests per decision
               Effect::None
           } else {
               Effect::None
           }
       }
       ```

    **Decision**: Per user - "Duplicate generation requests on same artifact silently ignored"

    **Pattern**: State transitions follow the decision flow:
    - Pending -> CheckSerialization -> (true) NeedsGeneration -> RunGenerator -> (success) -> Serialize -> (success) -> UpToDate
    - Pending -> CheckSerialization -> (true) NeedsGeneration -> RunGenerator -> (failure) -> Failed
    - Pending -> CheckSerialization -> (false) -> UpToDate

</action>
