    2. Clone the backend config and make config for use in spawn_blocking (these are cheap Arc clones or similar)

    3. Use `tokio::task::spawn_blocking` to execute `backend::serialization::run_check_serialization()`:
       - Pass the artifact reference, target, backend config, make config, and target_type
       - The function returns `Result<bool>` where bool indicates if generation is needed

    4. Convert the spawn_blocking result to EffectResult::CheckSerialization:
       - On Ok(needs_generation): return CheckSerialization { artifact_index, needs_generation }
       - On Err(e): return CheckSerialization { artifact_index, needs_generation: true } (fail open - assume generation needed on error)

    5. Ensure error handling preserves the error information - consider logging the error details

    **Important**: Use `spawn_blocking` because `run_check_serialization` executes a subprocess (check_serialization script) which is blocking I/O. Never call blocking operations directly in async context.

    **Pattern to follow**:
    ```rust
    EffectCommand::CheckSerialization { artifact_index, artifact_name, target, target_type } => {
        let backend = self.backend.clone();
        let make = self.make.clone();
        let result = tokio::task::spawn_blocking(move || {
            backend::serialization::run_check_serialization(
                &artifact_name, &target, &backend, &make, &target_type
            )
        }).await;
        
        match result {
            Ok(Ok(needs_generation)) => EffectResult::CheckSerialization { artifact_index, needs_generation },
            Ok(Err(_)) | Err(_) => EffectResult::CheckSerialization { artifact_index, needs_generation: true },
        }
    }
    ```

  <name>Task 2: Implement RunGenerator in Background Task</name>
  <files>pkgs/artifacts/src/background.rs</files>
  <action>
    Replace the stub RunGenerator implementation in `background::execute()` with real backend integration:

    1. Extract data from EffectCommand::RunGenerator (artifact_index, artifact_name, generator, prompts)

    2. Clone necessary config for spawn_blocking

    3. Create a temporary directory for generator output using `tempfile::TempDir::new()`:
       - This directory must survive the spawn_blocking call
       - Pass the path to the generator script

    4. Use `tokio::task::spawn_blocking` to execute `backend::generator::run_generator_script()`:
       - Pass: generator script path, prompts hashmap, output directory path
       - Returns: `Result<CapturedOutput>`

    5. Store the temp directory in the handler struct (add a field if needed) so it survives until Serialize effect:
       - Add `current_output_dir: Option<TempDir>` to BackgroundTask fields
       - Set it after successful generator execution

    6. Convert result to EffectResult::GeneratorFinished:
       - On Ok(captured_output): return GeneratorFinished { artifact_index, success: true, output: captured_output.to_string(), output_dir: temp_dir_path }
       - On Err(e): return GeneratorFinished { artifact_index, success: false, output: format!("Generator failed: {}", e), output_dir: None }

    7. Call `verify_generated_files()` after generator runs to ensure expected files were created

    **Critical**: The temp directory must not be dropped before Serialize runs. Store it in the handler struct.

    **Pattern**:
    ```rust
    EffectCommand::RunGenerator { artifact_index, artifact_name, generator, prompts } => {
        let temp_dir = tempfile::TempDir::new()?;
        let output_path = temp_dir.path().to_owned();
        let generator = generator.clone();
        let prompts = prompts.clone();
        
        let result = tokio::task::spawn_blocking(move || {
            backend::generator::run_generator_script(&generator, &prompts, &output_path)
        }).await;
        
        match result {
            Ok(Ok(output)) => {
                self.current_output_dir = Some(temp_dir); // Store to keep alive
                EffectResult::GeneratorFinished { 
                    artifact_index, 
                    success: true, 
                    output: output.to_string(),
                    output_dir: Some(output_path)
                }
            }
            Ok(Err(e)) => EffectResult::GeneratorFinished { artifact_index, success: false, output: e.to_string(), output_dir: None },
            Err(e) => EffectResult::GeneratorFinished { artifact_index, success: false, output: format!("Task panicked: {}", e), output_dir: None },
        }
    }
    ```
