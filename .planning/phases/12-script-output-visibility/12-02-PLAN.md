```rust
impl StepLogs {
    /// Get mutable reference to logs for a specific step
    pub fn get_mut(&mut self, step: LogStep) -> &mut Vec<LogEntry> {
        match step {
            LogStep::Check => &mut self.check,
            LogStep::Generate => &mut self.generate,
            LogStep::Serialize => &mut self.serialize,
        }
    }

    /// Append stdout lines as Output-level entries
    pub fn append_stdout(&mut self, step: LogStep, lines: &[String]) {
        let entries = lines
            .iter()
            .map(|line| LogEntry {
                level: LogLevel::Output,
                message: line.clone(),
            });
        self.get_mut(step).extend(entries);
    }

    /// Append stderr lines as Error-level entries
    pub fn append_stderr(&mut self, step: LogStep, lines: &[String]) {
        let entries = lines
            .iter()
            .map(|line| LogEntry {
                level: LogLevel::Error,
                message: line.clone(),
            });
        self.get_mut(step).extend(entries);
    }
}
```

**Rationale:**

- `get_mut` provides mutable access to specific step logs
- `append_stdout` adds lines with `LogLevel::Output` for stdout
- `append_stderr` adds lines with `LogLevel::Error` for stderr
- Methods handle the Vec<String> structure from CheckOutput/GeneratorOutput
