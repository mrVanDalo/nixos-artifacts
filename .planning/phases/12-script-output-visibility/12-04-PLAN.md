Modify background.rs to send OutputLine messages during script execution instead of waiting for completion.

**Find the function that runs generator/serialize scripts** (likely in background task execution).

**Current approach (blocking):**
- Script runs synchronously
- Full output collected at end
- Single EffectResult sent

**New approach (streaming):**

```rust
async fn run_script_streaming(
    artifact_index: usize,
    command: &mut tokio::process::Command,
    tx: &mpsc::UnboundedSender<EffectResult>,
) -> Result<ScriptOutput, String> {
    use tokio::io::AsyncBufReadExt;
    
    // Spawn the process
    let mut child = command
        .stdout(std::process::Stdio::piped())
        .stderr(std::process::Stdio::piped())
        .spawn()
        .map_err(|e| format!("Failed to spawn: {}", e))?;
    
    // Take ownership of stdout/stderr
    let stdout = child.stdout.take().ok_or("Failed to get stdout")?;
    let stderr = child.stderr.take().ok_or("Failed to get stderr")?;
    
    // Create async readers
    let mut stdout_reader = tokio::io::BufReader::new(stdout).lines();
    let mut stderr_reader = tokio::io::BufReader::new(stderr).lines();
    
    let mut stdout_lines = Vec::new();
    let mut stderr_lines = Vec::new();
    
    // Stream lines as they arrive
    loop {
        tokio::select! {
            // Check if stdout has a line
            line = stdout_reader.next_line() => {
                match line {
                    Ok(Some(line)) => {
                        stdout_lines.push(line.clone());
                        tx.send(EffectResult::OutputLine {
                            artifact_index,
                            stream: OutputStream::Stdout,
                            content: line,
                        }).ok();
                    }
                    Ok(None) => {} // Stream closed
                    Err(e) => {
                        let _ = tx.send(EffectResult::OutputLine {
                            artifact_index,
                            stream: OutputStream::Stderr,
                            content: format!("stdout read error: {}", e),
                        });
                    }
                }
            }
            // Check if stderr has a line
            line = stderr_reader.next_line() => {
                match line {
                    Ok(Some(line)) => {
                        stderr_lines.push(line.clone());
                        tx.send(EffectResult::OutputLine {
                            artifact_index,
                            stream: OutputStream::Stderr,
                            content: line,
                        }).ok();
                    }
                    Ok(None) => {} // Stream closed
                    Err(e) => {
                        let _ = tx.send(EffectResult::OutputLine {
                            artifact_index,
                            stream: OutputStream::Stderr,
                            content: format!("stderr read error: {}", e),
                        });
                    }
                }
            }
            // Check if process completed
            status = child.wait() => {
                return match status {
                    Ok(status) if status.success() => {
                        Ok(ScriptOutput { stdout_lines, stderr_lines })
                    }
                    Ok(status) => {
                        Err(format!("Process exited with code: {:?}", status.code()))
                    }
                    Err(e) => {
                        Err(format!("Wait failed: {}", e))
                    }
                };
            }
        }
    }
}
```

**Integrate into existing functions** - modify run_generator or similar to use streaming:

```rust
pub async fn run_generator(
    artifact_index: usize,
    // ... other params
    tx: mpsc::UnboundedSender<EffectResult>,
) {
    let mut cmd = tokio::process::Command::new("/path/to/generator");
    // ... setup environment ...
    
    match run_script_streaming(artifact_index, &mut cmd, &tx).await {
        Ok(output) => {
            tx.send(EffectResult::GeneratorFinished {
                artifact_index,
                success: true,
                output,
                error: None,
            }).ok();
        }
        Err(e) => {
            tx.send(EffectResult::GeneratorFinished {
                artifact_index,
                success: false,
                output: ScriptOutput::default(),
                error: Some(e),
            }).ok();
        }
    }
}
```

**Key changes:**
1. Use `tokio::process::Command` instead of `std::process::Command`
2. Stream stdout/stderr lines via `AsyncBufReadExt::lines()`
3. Send `EffectResult::OutputLine` for each line as it arrives
4. Collect lines for final `ScriptOutput` returned at end
5. Process exit status still captured

**Dependencies:** Ensure tokio is available with the "process" feature.
