<task type="auto">
  <name>Task 2: Update model builder to set error status</name>
  <files>pkgs/artifacts/src/tui/model_builder.rs</files>
  <action>
Update build_model to handle validation errors:
1. When creating SharedEntry in the entries vector
2. Check if shared.info.error is Some
3. If error exists:
   - Set status to ArtifactStatus::Failed with:
     - error: the validation error message
     - output: empty string
     - retry_available: false (validation errors don't benefit from retry)
4. If no error, status remains Pending (will be checked normally)

The code should be in the loop that adds shared artifacts:

```rust
for (_name, shared_info) in shared_artifacts {
    let status = if let Some(ref error) = shared_info.error {
        ArtifactStatus::Failed {
            error: error.clone(),
            output: String::new(),
            retry_available: false,
        }
    } else {
        ArtifactStatus::Pending
    };
    
    entries.push(ListEntry::Shared(SharedEntry {
        info: shared_info,
        status,
        step_logs: StepLogs::default(),
        selected_generator: None,
    }));
}
```

</action>
