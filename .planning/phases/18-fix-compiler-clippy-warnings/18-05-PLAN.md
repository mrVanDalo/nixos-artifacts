## Pedantic and Nursery Lints

Clippy has stricter lint categories beyond default:
- `clippy::pedantic` - Strict style lints
- `clippy::nursery` - Experimental/new lints

**Important:** NOT all pedantic/nursery lints need to be fixed. The requirement says "addressed where appropriate" - some lints may be too strict for this codebase.

**Strategy:**
1. Run with pedantic + nursery to see what warnings exist
2. Review each warning category
3. Fix reasonable ones, add `#[allow(...)]` with justification for ones that don't fit
4. Document the decision

**Common pedantic warnings that may need discussion:**
- `must_use_candidate` - Suggests adding `#[must_use]` to many functions
- `missing_errors_doc` - Wants `# Errors` sections in docs
- `missing_panics_doc` - Wants `# Panics` sections in docs
- `module_name_repetitions` - Warns on `foo::Foo` naming
- `similar_names` - Warns on similar variable names
- `too_many_lines` - Warns on long functions
- `type_complexity` - Warns on complex types

**Common nursery warnings:**
- Various experimental lints that may be noisy
</context>
