= Backend Quickstart
:description: Create a working backend for nixos-artifacts in 10 minutes

Create a custom backend with minimal boilerplate. A backend is a set of scripts that store artifact files generated by nixos-artifacts.

== Minimal Backend Structure

[source]
----
my-backend/
├── backend.toml           # Backend configuration
├── check.sh               # Check if artifact exists (unified)
└── serialize.sh           # Store artifact files (unified)
----

With the unified interface, a single script can handle all target types (NixOS, HomeManager, and shared).

== backend.toml

[source,toml]
----
[mybackend.nixos]
enabled = true
check = "./check.sh"
serialize = "./serialize.sh"

[mybackend.home]
enabled = true
check = "./check.sh"
serialize = "./serialize.sh"

[mybackend.shared]
enabled = true
check = "./check.sh"
serialize = "./serialize.sh"

[mybackend.settings]
# Optional: backend-specific configuration (available in targets.json)
storage_path = "/var/lib/secrets"
----

== Unified Environment Variables

All scripts receive the same environment variables:

[cols="1,2,3"]
|===
| Variable | Type | Description

| `$artifact` | String | Artifact name
| `$artifact_context` | String | `"nixos"`, `"homemanager"`, or `"shared"`
| `$targets` | File | Path to targets.json with all target info
| `$inputs` | Directory | File metadata JSONs (check only)
| `$out` | Directory | Generated files (serialize only)
|===

== check.sh

Check if an artifact already exists (skip generation if it does):

[source,bash]
----
#!/usr/bin/env bash
set -euo pipefail

# Unified check script - works for all contexts
# Environment: $artifact, $artifact_context, $targets, $inputs

# Check all targets
jq -c '.targets[]' "$targets" | while read -r target; do
    name=$(echo "$target" | jq -r '.name')
    storage_path=$(echo "$target" | jq -r '.config.storage_path // "/var/lib/secrets"')

    for meta in "$inputs"/*; do
        [ -f "$meta" ] || continue
        filename=$(basename "$meta")
        [ -f "$storage_path/$name/$artifact-$filename" ] || exit 1
    done
done

echo "EXISTS"
exit 0
----

Exit codes:

* `0` — Artifact exists, skip generation
* Non-zero — Needs generation

== serialize.sh

Store files for all targets:

[source,bash]
----
#!/usr/bin/env bash
set -euo pipefail

# Unified serialize script - works for all contexts
# Environment: $artifact, $artifact_context, $targets, $out

# Process each target
jq -c '.targets[]' "$targets" | while read -r target; do
    name=$(echo "$target" | jq -r '.name')
    storage_path=$(echo "$target" | jq -r '.config.storage_path // "/var/lib/secrets"')

    mkdir -p "$storage_path/$name"

    for file in "$out"/*; do
        [ -f "$file" ] || continue
        filename=$(basename "$file")
        cp "$file" "$storage_path/$name/$artifact-$filename"
    done
done
----

== Test Your Backend

.Make scripts executable:
[source,bash]
----
chmod +x my-backend/*.sh
----

.Run with your flake:
[source,bash]
----
NIXOS_ARTIFACTS_BACKEND_CONFIG=/path/to/my-backend/backend.toml \
  artifacts /path/to/flake
----

== targets.json Format

The `$targets` file contains all target information:

For single NixOS target:
[source,json]
----
{
  "context": "nixos",
  "targets": [
    {
      "name": "server-one",
      "type": "nixos",
      "config": {"storage_path": "/var/lib/secrets"}
    }
  ]
}
----

For single HomeManager target:
[source,json]
----
{
  "context": "homemanager",
  "targets": [
    {
      "name": "alice@workstation",
      "type": "homemanager",
      "config": {"storage_path": "~/.secrets"}
    }
  ]
}
----

For shared artifacts:
[source,json]
----
{
  "context": "shared",
  "targets": [
    {"name": "server-one", "type": "nixos", "config": {...}},
    {"name": "server-two", "type": "nixos", "config": {...}},
    {"name": "alice@host", "type": "homemanager", "config": {...}}
  ]
}
----
